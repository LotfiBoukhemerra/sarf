يحتوي قرص نظام الاشتقاق والتصريف على المجلد (Arabic Morphology System) والملف Readme العربي والإنكليزي.
يتضمن المجلد (Arabic Morphology System) المجلدات الآتية:
1- المجلد program : وفيه :
	- النسخة التنفيذية لبرنامج النظام Sarf.jar  
	- مجلد قواعد المعطيات (db) 
	- مجلد الملفات المساعدة (documents)
2- المجلد src : وفيه البرنامج المصدري (source program) للنظام.
3- المجلد doc : وفيه وثائق المشروع (أربع ملفات لكراسات النظام).
4- المجلد util: وفيه النسخة التنفيذية لآلة جافا jdk-1_5_0_01-windows-i586-p.exe


متطلبات تشغيل النظام :
  يتطلب تشغيل النظام وجود الآلة الافتراضية لجافا في نظام التشغيل، لذا يجب تشغيل
 النسخة التنفيذية jdk - الموجودة في المجلد util - عند تشغيل النظام أول مرة فقط.

يمكن تشغيل النظام من القرص مباشرة.
ويمكن نسخ النظام على أي مسوق (drive) في الحاسوب [ :C أو :D أو...]، ثم تشغيله من هذا المسوق.


العرض: 
يجب أن تكون دقة الشاشة (screen resolution (1024 * 768 للحصول على أفضل مساحة للعرض.
