Welcome to Sarf - The Arabic Language Morphology System

The goal of this program is to display the various derivatives of an Arabic root word. A current limitation is that the program cannot extract the root by itself, so you must enter the root.
Hopefully, in the future root extraction (stemming) will be added so all you have to do is enter a word.

I hope you find will this program useful. If you have comments or questions, contact the developer here https://github.com/alsaydi/sarf (Feel free to open an issue).
